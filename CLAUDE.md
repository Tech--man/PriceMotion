## Project Overview

PriceMotion is a Chinese LLM API pricing comparison site ‚Äî a single-page app that displays pricing data for domestic (Chinese) large language model APIs side by side. All text is in Chinese (zh-CN). Data is updated daily via a Playwright-based scraper running on GitHub Actions.

## Architecture

No build step, no framework. Static files served via GitHub Pages.

### Data Pipeline

```
providers.json ‚îÄ‚îÄ‚îê
                 ‚îú‚îÄ‚Üí scrape.py ‚îÄ‚îÄ‚Üí models.json ‚îÄ‚îÄ‚Üí index.html (fetch)
previous models ‚îÄ‚îÄ‚îò    (Playwright)    (committed)     (browser)
```

- **`index.html`** ‚Äî Self-contained SPA. Inline CSS (dark/light theme via `.dark` class + CSS variables), inline JS IIFE. Loads `models.json` via `fetch()`. Renders table and card grid views with client-side filtering, sorting, search. Theme persisted in `localStorage`.
- **`scrape.py`** ‚Äî Playwright-based scraper. Launches headless Chromium, renders JS-heavy provider pages, extracts pricing tables. Merges scraped prices with editorial metadata from `providers.json`. Preserves models from previous `models.json` when a provider scrape fails. Outputs `models.json`.
- **`providers.json`** ‚Äî Editorial metadata that cannot be scraped: company names, colors, model descriptions, tags, notes, context window sizes. Updated manually when new models are added.
- **`models.json`** ‚Äî Generated by `scrape.py`, committed to repo, served by GitHub Pages. Contains full model objects with `lastUpdated` timestamps.
- **`.github/workflows/scrape.yml`** ‚Äî GitHub Actions workflow. Runs daily at 02:00 UTC (10:00 AM CST) and on manual trigger. Installs Playwright+Chromium, runs scraper, validates output, commits `models.json` if changed.

### 9 Provider Scrapers

Each scraper in `scrape.py` receives a Playwright `Browser` instance, navigates to the provider's pricing page, waits for dynamic content to render, then extracts model data from tables or card layouts. Providers: DeepSeek, Qwen, Zhipu/GLM, Moonshot/Kimi, Baichuan, Xfyun/Spark, Doubao, Hunyuan, ERNIE.

## Development

### Frontend

No build tools needed. Test locally:
```bash
python -m http.server
# Open http://localhost:8000
```

### Scraper

```bash
pip install -r requirements.txt
playwright install chromium
python scrape.py              # outputs models.json
```

### Trigger scraper manually (GitHub Actions)

Go to Actions tab ‚Üí "Daily Price Scrape" ‚Üí "Run workflow".
